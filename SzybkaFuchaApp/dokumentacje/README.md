# Dokumentacje

Ten folder zawiera bieżące dokumentacje operacyjne projektu.

Aktualnie:

- `automatyzacja-zdjec-openclaw.md` — dokładny opis tworzenia obrazów do postów przez OpenClaw, `Replicate` i skrypty bloga
- `optymalizacja-postow-seo-aeo-cwv.md` — porządki w szablonach postów i wdrożone optymalizacje SEO, AEO, GEO, Core Web Vitals i dostępności
- `audyt-lighthouse-i-optymalizacja-obrazow.md` — automatyczny audit Lighthouse, generowanie `webp/avif` i sposób działania tego pipeline w OpenClaw oraz GitHub Actions
- `bielik-lokalnie-openclaw.md` — rekomendowany lokalny model Bielik dla Intel Maca, wybór GGUF, uruchomienie przez Ollama i bezpieczne użycie w OpenClaw tylko do generowania treści
- `Bielik-4.5B-Modelfile` — gotowy `Modelfile` do utworzenia lokalnego modelu `bielik-content` w Ollamie
